# OneSec Key Token Bridge

The Key Token Bridge enables seamless transfer of any token across any
blockchain, from one chain to another.

The project documentation is in `cargo doc`.

Run the following to generate it:

```
cargo doc -p one_sec --no-deps
```

After than open `target/doc/one_sec/index.html` in the browser.

You can also browse [the pre-generated docs](https://luxbj-aqaaa-aaaah-qqbdq-cai.icp0.io/doc/one_sec/index.html).
